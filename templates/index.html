Altitude Performance Conversion Calculator







The altitude performance conversion calculator was developed by researchers from Indiana University as part of a dissertation project in collaboration with the NCAA and DirectAthletics using the TFRRS database.We combined the indoor track and field results database with geographical survey data to establish competition elevations. By observing the performance changes of each athlete within a given NCAA indoor track season we were able to model the percent change from sea level (1-150m) for each event (800m-5,000m) for males and females and extrapolate the pattern to longer events using physiological principals.
